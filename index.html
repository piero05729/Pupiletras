    ¿Cómo jugar?

    Pulsa en una letra para iniciar y arrastra en línea recta hasta la última letra. Suelta para validar. Se aceptan 8 direcciones. El mismo enlace (semilla) genera el mismo tablero para todos.

    Compartir

    Usa el botón "Copiar enlace" o comparte la URL con los parámetros ?seed=...&size=...
